# Weekly Student Expense Tracker

##  Description

This program is a simple Python-based expense tracker designed for students. It allows the user to input their weekly budget and record up to four expenses under different categories such as food, transportation, and more.

The program calculates the total amount spent, the remaining balance, and determines whether the user stayed within their budget. It also highlights any expense that exceeds 25% of the weekly budget.

##  Features

* Accepts student name and weekly budget
* Displays 5 expense categories
* Allows up to 4 expense entries (with option to skip)
* Validates user input (rejects invalid or empty inputs)
* Flags high expenses (more than 25% of budget)
* Shows a complete expense report with status

##  How to Run

1. Make sure Python is installed on your computer
2. Download or clone this repository
3. Open a terminal in the project folder
4. Run the program using:

```
python your_file_name.py
```

5. Follow the on-screen instructions

##  Output

The program will display:

* Student name and weekly budget
* List of recorded expenses
* Total amount spent
* Remaining balance
* Budget status (OK or Overspent)

##  Notes

* Only whole numbers are accepted for budget and expenses
* Invalid inputs will prompt the user to try again
